# Kar


KaradisBrowser — это веб-обозреватель, разработанный на платформе WPF. Для отображения веб-страниц интегрирована библиотека CefSharp, а логика взаимодействия с графическим интерфейсом построена с использованием привязанного к разметке кода. Проект поддерживает базовые функции навигации в сети и включает внутренние страницы для начального экрана и управления параметрами.

Основные возможности

    Работа с несколькими вкладками (создание, переключение, закрытие).

    Встроенная начальная страница с собственным оформлением.

    Раздел пользовательских настроек с сохранением данных в формате JSON.

    Обработка всплывающих окон и контекстных меню через переопределенные обработчики жизненного цикла.

    Конвертация и отображение адресов веб-страниц.

Применяемые технологии

    Основной язык программирования: C#

    Построение оконного интерфейса: Windows Presentation Foundation

    Движок отображения страниц: CefSharp (на базе Chromium)

    Языки разметки и стилизации внутренних компонентов: HTML, CSS, JavaScript

Структура исходного кода

    Каталог Kar — содержит основную часть программного кода и разметки.

    Файл Kar.slnx — файл решения для открытия проекта в среде разработки.

    Каталог Assets/Icons — набор векторных изображений для кнопок управления.

    Каталоги Homepage и Settings — веб-файлы для соответствующих внутренних страниц приложения.

    Файлы с расширением xaml и cs — элементы окон, вкладок и привязанный к ним программный код.

    Файл LICENSE.txt — условия распространения программного обеспечения.
